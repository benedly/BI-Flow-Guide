# API

### enable & disabled

```javascript
// @param idx -> tab Index Number
// @param flag -> true(default) or false
comp.enable(idx, flag);
comp.disabled(idx, flag);
```

### showTab & hideTab

```javascript
// @param idx -> tab Index Number
comp.showTab(idx);
comp.hideTab(idx);
```

### setSelection

```javascript
// @param idx -> tab Index Number
comp.setSelection(idx);
```
